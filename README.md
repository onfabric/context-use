# context-use

## Development

To run the project, use the following command:

```bash
bun dev
# or also: bun run dev
```

To build the project, use the following command:

```bash
bun run build
```

## Code Style

To check the code style, use the following command:

```bash
bun run codestyle:check
```

To fix the code style, use the following command:

```bash
bun run codestyle:fix
```
